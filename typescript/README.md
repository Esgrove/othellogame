# Othello TypeScript

TypeScript implementation using npm or Deno.

## Build and run

Using npm scripts:

```shell
npm run build
npm run start
```

Directly with Node:

```shell
node src/othello.ts --help
```

Deno 2.0:

```shell
deno install
deno run --allow-env --allow-sys src/othello.ts --help
```
