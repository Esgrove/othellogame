# Othello TypeScript

TypeScript implementation using Bun.

## Build and run

Using scripts:

```shell
bun run build
bun othello
bun othello --help
```

Directly with Bun:

```shell
bun src/othello.ts --help
```

## Tests

Tests use the built-in [Bun test runner](https://bun.sh/docs/cli/test).

```shell
bun test
```
